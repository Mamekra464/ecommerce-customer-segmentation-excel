# E-commerce Customer Segmentation & Sales Dashboard (Excel)

## Overview
This project analyzes customer purchasing behavior using Excel. It includes data cleaning, statistical analysis, pivot tables, Goal Seek, and an interactive dashboard.

## Key Features
- Data cleaning and preparation
- Mean, median, and mode calculations
- Goal Seek analysis
- Pivot tables with slicers
- Interactive dashboard creation
- Data visualization using charts
- Customer segmentation analysis

## Skills Demonstrated
- Excel Data Analysis
- Dashboard Development
- Pivot Tables and Slicers
- Goal Seek
- Customer Segmentation
- Business Intelligence
